# Meridian Voice Concierge

Voice concierge MVP for The Meridian Casino & Resort.

## Planned stack
- Backend API: .NET
- Voice agent: TypeScript with LiveKit
- Database: PostgreSQL
- Infra: Docker Compose

## Project structure
- `src/api/Meridian.Api` - backend API
- `src/agent` - voice agent
- `docs` - supporting docs

## Status
Phase 1 complete: repository and project foundation created.
